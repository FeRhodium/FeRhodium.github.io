Limit and Continuity

very rough Def: $f(x)$ has a limit $L$ as $x$ tends to $c$, if $f(x)$ tends to $L$ as $x$ tends to $c$. We write $\lim_{x\to c} f(x)=L$

Ex 1: $f(x)=2024x+916$, $\lim_{x\to 0}f(x)=916$

Ex 2: $\lim_{x\to 0}\dfrac{2024x+916}{215+x^2}=\dfrac{916}{215}$

Motivation: Why limit?

  1. Instantaneous speed of moving object

    • object moving on the y-axis positive direction, whose position fuc is $y = s(t)$, where $t$ in sec, $s(t)$ in meters

      • $s(t)=t^2$, get the instantaneous speed $v(t)$ at $t=1$

        Idea: consider small time interval $[1, 1 + h]$ if $ h > 0$, $[1 + h, 1]$ if $h < 0$.

        Average speed in time interval $[1, 1 + h] = \dfrac{dis}{time}=\dfrac{s(1+h) - s(1)}{h}$

        $v(1)=\lim_{h \to 0}\dfrac{s(1 + h) - s(1)}{h} = \lim_{h\to 0}\dfrac{(1 + h)^2 - 1^2}{h} = \dfrac 00$? it is an indeterminate forms, which is a devil.

        $v(1)=\lim_{h \to 0}\dfrac{(1 + h - 1)(1 + h + 1)}h = \lim_{h\to 0}2 + h=2$

    instantaneous speed $v(t_0)=\lim_{h\to 0}\dfrac{s(t_0 + h) - s(t_0)}h$, the top means change in $y$, the down means change in $t$, which can be written in $\lim_{\Delta t\to 0}\dfrac{\Delta y}{\Delta t}$

  2. slope of the tangent line of curves, when tangent means "to touch" in Latin.

    • T-line at $(0, 0)$ when the curve is $y = x^3$, but it cut through?

    • Why care about tangent line?

      it is the best line to approximate a curve, which can be seperated by lots of T-lines.

    • 300 BC Euclid: T-line a line s.t. no ther straight lines for falling betweent curve and T-line.

    • Nov 13, 1665 Newton slope $=\frac qp$.

      • Ex: $y=x^2$ T-line at $(1, 1)$, and another point is $(1 + o, 1 + o\cdot\frac pq)$, where $o$ is a tiny amount. so we have $1 + o\cdot\frac pq = (1 + o)^2$, which can be solve that $\frac pq = 2$, represents the slope at $(1, 1)$

    • Leibnitz: T-line is the line through a pair if infinitely close points on curve $=\lim_{Q \to P}$ secant line, so the slope of T-line $=\lim_{Q\to P}$ slope of secant line.

      • $y = f(x)$, slope at $P(c, f(c))$, assuming a point $Q(c + h, f(c + h))$, slope $=\lim_{h\to 0}\dfrac{f(c + h) - f(c)}h=\lim_{\Delta x\to 0}\dfrac{\Delta y}{\Delta x}$

  3. Let $y = f(x)$ be a general function, average rate of change of $f$ over $[x_1, x_2]$ $=\dfrac{f(x_2) - f(x_1)}{x_2 - x_1}$, where instantaneous rate of change of $f$ at $x_1$ $=\lim_{x_2\to x_1}\dfrac{f(x_2) - f(x_1)}{x_2 - x_1}=\lim_{h \to 0}\dfrac{f(x_1 + h) - f(x_1)}{h}$

More formal def of limits

Let $f(x)$ be defined in an open interval containing $c$, possibly not at $c$ itself, which can be truncated neighborhood of $c$. If $f(x)$ is arbitrarily close to the $L$, as close as we wish, as long as $x$ is close but not to equal to $c$ enough, then we say $f$ approaches the limit $L$ as $x\to c$.

Notation: $\lim_{x\to c}f(x) = L$

Does $\lim_{x\to c} f(x)$ alwats exists?

Of course not.

  1. $f(x)=\begin{cases} 0 & x < 0\ 1 &x \ge 0\end{cases}$, $\lim_{x\to 0}f(x)$ DNE, since the left and right limit are not equal.
  2. $f(x)=\sin\left(\dfrac 1x\right)$, $\lim_{x\to 0}f(x)$ DNE, which is too crazy. (a graph should be here?)
  3. $f(x)=\dfrac{1}{x^2}$, $\lim_{x\to 0}f(x)$ DNF, which can be inf.

Limit laws

if $\lim_{x\to c}f(x)$ and $\lim_{x\to c}g(x)$ exist. Then

  1. $\lim_{x\to c}(f(x) + g(x)) = \lim_{x\to c}f(x) + \lim_{x\to c}g(x)$

  2. suppose $k$ is a constant, $\lim_{x\to c}kf(x) = k\lim_{x\to c}f(x)$

    so we have $\lim_{x\to c}(c_1f(x) + c_2g(x)) = c_1\lim_{x\to c}f(x) + c_2\lim_{x\to c}g(x)$.

  3. $\lim_{x\to c}(f(x)g(x)) = \lim_{x\to c}f(x)\lim_{x\to c}g(x)$

  4. $\lim_{x\to c}\dfrac{f(x)}{g(x)} = \dfrac{\lim_{x\to c}f(x)}{\lim_{x\to c}g(x)}$, when $\lim_{x\to c}g(x)\neq 0$

  5. $\lim_{x\to c}f^n(x)=\left(\lim_{x\to c}f(x)\right)^n$

Ex: when $f(x)$ is a polynomial function, then $\lim_{x\to c}f(x) = f(c)$
